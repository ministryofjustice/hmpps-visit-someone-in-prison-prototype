Accessibility statement for the ‘visit someone in prison’ service

This accessibility statement applies to prison-visits.service.justice.gov.uk

This website is run by HM Prison and Probation Service (HMPPS).

We want as many people as possible to be able to use this website. We will make improvements to accessibility as we find issues.

AbilityNet has advice on making your device easier to use if you have access needs.

Compliance status

This website is fully compliant with the Web Content Accessibility Guidelines version 2.2 AA standard.

It was last tested by User Vision in October 2024 against the WCAG 2.2 AA standard.

User Vision assessed pages using the following assistive technology:

  • NVDA (using Edge, Chrome and Firefox browsers)
  • JAWS (using Edge, Chrome and Firefox browsers)
  • TalkBack for Android
  • VoiceOver for iOS and MacOS
  • Dragon Naturally Speaking on desktop

This testing identified 3 non-compliance issues. These issues were fixed on [date]. They were tested to check compliance with the WCAG 2.2 AA standard by our website team.

Feedback

If you find any problems or think we’re not meeting accessibility requirements, you can send us feedback.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this website’s accessibility

HMPPS is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Preparation of this accessibility statement

This statement was prepared on 29 May 2024. It was last reviewed on [date this version is published].
